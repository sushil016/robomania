# Event Images Guide

## Required Images

Please add the following images to the `/public` folder:

### 1. robowar.png
- **Dimensions**: Recommended 800x600px or similar aspect ratio (4:3 or 16:9)
- **Content**: Should show robot combat, battle arena, or warrior robots
- **Style**: Action-packed, intense, energetic
- **Colors**: Orange/Red tones work best with the theme

### 2. roborace.png
- **Dimensions**: Recommended 800x600px or similar aspect ratio (4:3 or 16:9)
- **Content**: Should show racing robots, speed tracks, or fast-moving bots
- **Style**: Dynamic, fast-paced, sleek
- **Colors**: Cyan/Blue tones work best with the theme

### 3. robosoccer.png
- **Dimensions**: Recommended 800x600px or similar aspect ratio (4:3 or 16:9)
- **Content**: Should show robots playing soccer, soccer field, or bots with balls
- **Style**: Sporty, teamwork-focused, competitive
- **Colors**: Green tones work best with the theme

## Image Specifications

- **Format**: PNG (preferred) or JPG
- **Size**: Keep under 500KB for optimal loading
- **Optimization**: Use tools like TinyPNG or ImageOptim to compress
- **Quality**: High-resolution but web-optimized

## Placement

All images should be placed directly in:
```
/public/robowar.png
/public/roborace.png
/public/robosoccer.png
```

The images will automatically appear in:
- Event details page cards
- Event section on landing page
- Event modal headers

## Fallback

If images are not yet available, the component will show a gradient background with the event icon in the center.
